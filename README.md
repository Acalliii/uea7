##  Objetivo del repositorio 

La principal función de este repositorio es organizar y almacenar los ejercicios, pruebas y códigos realizados en el transcurso de esta materia “Producción y Comunicación en la Cultura Colaborativa”. Estos ejercicios son desarrollados en Visual Studio Code utilizando lenguajes como HTML, CSS, JavaScript y JSON, esto permite mantener un registro claro y ordenado de cada actividad trabajada durante el trimestre. A su vez, esta herramienta ubicada en GitHub ayuda con el seguimiento de los ejercicios realizados a lo largo del curso, permitiendo que cada avance, corrección o actualización quede documentado de manera precisa en tiempo y forma. 

De esta manera, el repositorio se convierte en un espacio accesible tanto para mí, al momento de revisar  o consultar ejercicios previos, como para el profesor, quien puede analizar el progreso,dar retroalimentación y asignar calificaciones con base en lo que se encuetra visible en el repositorio. Este espacio en la web funciona como archivo dinámico donde se registran prototipos, experimentos, errores y soluciones, lo que me ayuda a visualizar de forma más amplia el proceso de aprendizaje técnico. En resumidas cuentas el propósito se puede resumir en que este funciona como una herramienta que apoya la continuidad, la recuperación de materiales y el fortalecimiento de las competencias digitales necesarias para el desarrollo colaborativo dentro del ámbito artístico.

## Organización del repositorio

La organización del repositorio está pensada para que, al ingresar, sea posible localizar los códigos y ejercicios de manera clara, rápida y sencilla. GitHub facilita esto gracias a sus herramientas para crear carpetas y subcarpetas, estas permiten agrupar cada ejercicio dentro del apartado correspondiente y mantener una distribución lógica del contenido. Cada vez que inicio un nuevo proyecto, genero una carpeta principal que contiene los archivos esenciales, usualmente HTML,JavaScript y Jason, en algunas ocaciones si el proyecto requiere recursos como imágenes, audios o videos estos se integran en subcarpetas específicas llamadas assets, estas ayudan a mantener los elementos organizados para su funcionamiento adecuado. 

Este orden asignado permite que el repositorio mantenga una navegación funcional en cuanto a los contenidos creados a lo largo de las clases de este curso. Por su parte el que los elementos de cada nuevo proyecto se encuentren divididos en carpetas individuales y no esten todos juntos facilita el entendimiento, la visualización y la relación entre cada archivo.  
En conclusión todo lo anterior me permite subir los contenidos de los proyectos de manera sencilla y organizada a mi repositorio de Github. 


## Experiencia personal de crear y gestionar el repositorio

Mi experiencia utilizando GitHub y gestionando mis repositorios se ha convertido en un proceso formativo esencial dado a lo largo de varios trimestres. No solo ha fortalecido mi capacidad de organización, sino también mi comprensión del trabajo tecnológico y del flujo digital que implica crear, documentar y mantener un proyecto artistico. Al inicio, la plataforma me resultaba confusa, especialmente al crear carpetas, decidir qué iba dentro de cada una o entender funciones como clonar y compartir repositorios. Sin embargo, con la práctica constante y el avance en la carrera, su uso se ha vuelto cada vez más sencillo. Nombrar proyectos, crear carpetas, realizar forks ahora son parte de una rutina que me brinda un sentido de control y libertad creativa.

GitHub ha facilitado tanto mi trabajo individual como el colaborativo. En proyectos personales, la organización de los repositorios me permite mantener un orden claro de mis avances, recursos, experimentos y pruebas. En cuanto a los trabajos grupales, las herramientas de colaboración como lo son  organizaciones y los repositorios compartidos hacen que la comunicación se vuelva casi automática, permitiendo que cada integrante contribuya de forma activa.
Hoy, gracias al uso combinado de GitHub y Visual Studio Code, puedo desarrollar proyectos de manera organizada, accesible y clara, algo que considero fundamental para mi formación académica y artistica.
